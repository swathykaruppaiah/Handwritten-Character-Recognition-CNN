# Handwritten Character Recognition Using CNN

## Project Description
This project uses Convolutional Neural Networks (CNN) to recognize handwritten characters and digits from image datasets. The model is trained using deep learning techniques to achieve high accuracy in character recognition.

## Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib

## Features
- Handwritten digit recognition
- CNN-based deep learning model
- High accuracy prediction
- Image preprocessing and classification

## Dataset
MNIST handwritten digit dataset

## Output
The system predicts handwritten digits from input images using a trained CNN model.

## Conclusion
The project demonstrates how CNN can be used effectively for handwritten character recognition tasks in Artificial Intelligence and Machine Learning.
